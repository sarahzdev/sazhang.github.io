Just because we have these fancy sectioning elements now, it doesn’t mean that the good old div element has lost all of its usefulness. div has no semantic meaning, so it is quite useful whenever we are altering the HTML purely for the sake of styling purposes. Let’s say that we have a blog post contained inside an article element that we need to wrap in something for the sake of styling purposes. Reaching for the section element in this circumstance is not the right thing to do. Though section is technically a generic element, div is the far more appropriate option in this circumstance. This new wrapping container is not meant to have any semantic meaning behind it and that is exactly what div is designed to be used for.
